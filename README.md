# TTC Bus Delay Dashboard

Interactive Power BI dashboard and written report analyzing TTC bus delay records from January to June 2022.

## Project Files

- `index.html` - hosted project page with the embedded dashboard
- `TTC_Delay_Dashboard.pbix` - Power BI dashboard file
- `logo.png` - TTC logo used on the hosted page

## Key Findings

- Route 36 had the highest number of recorded delay incidents.
- Saturday and Wednesday appeared as peak delay days.
- Cleaning and Disinfection had the longest average-delay incident categories after standardizing incident labels.

## Tools

Power BI, Python, Pandas, NumPy, Matplotlib, and Seaborn.
